🚀 INSTALAÇÃO RÁPIDA (5 PASSOS):

1️⃣ EXTRAIR ARQUIVO
   • Baixe: NAIPERS-CLUB-COMPLETO.tar.gz
   • Extraia: naipers-club-final/
   • Terá todos os arquivos organizados

2️⃣ UPLOAD PARA HOSTINGER
   • cPanel → File Manager
   • Navegue até: public_html/
   • Upload de TODA a pasta naipers-club-final/
   • Ou mova o conteúdo de naipers-club-final/ para public_html/

3️⃣ CRIAR E IMPORTAR BANCO
   • cPanel → MySQL Databases → Create
   • Anote: nome, usuário, senha
   • phpMyAdmin → Import → schema_mysql.sql
   • Verificar 3 tabelas criadas ✅

4️⃣ CONFIGURAR CREDENCIAIS
   • Edite: config/database.php
   • Preencha credenciais MySQL
   • Troque senha de admin!
   • Salve arquivo

5️⃣ TESTAR TUDO
   • https://seudominio.com (quiz)
   • https://seudominio.com/admin (dashboard)
   • Faça um teste completo do quiz
   • Verifique se leads aparecem no admin
