🔧 1. CREDENCIAIS DO BANCO MySQL:

Abra: config/database.php

Encontre estas linhas:

┌──────────────────────────────────────────────────────────────┐
│ define('DB_HOST', 'localhost');                              │
│ define('DB_NAME', 'seu_usuario_naipers_club');  ← TROCAR    │
│ define('DB_USER', 'seu_usuario_mysql');         ← TROCAR    │
│ define('DB_PASS', 'sua_senha_mysql');           ← TROCAR    │
└──────────────────────────────────────────────────────────────┘

O QUE COLOCAR:

• DB_HOST: 
  → Hostinger: 'localhost' (não mude!)
  
• DB_NAME: 
  → Nome completo do banco que você criou
  → Exemplo: 'u123456_naipers_club'
  → No Hostinger geralmente é: seu_usuario_nome_banco
  
• DB_USER:
  → Usuário MySQL que você criou
  → Exemplo: 'u123456_admin'
  
• DB_PASS:
  → Senha do usuário MySQL
  → A que você definiu ao criar o usuário

──────────────────────────────────────────────────────────────────

🔐 2. CREDENCIAIS DE ADMIN (Dashboard):

No mesmo arquivo: config/database.php

Encontre estas linhas:

┌──────────────────────────────────────────────────────────────┐
│ define('ADMIN_USERNAME', 'admin');                           │
│ define('ADMIN_PASSWORD', password_hash('naipersadmin2024',  │
│        PASSWORD_BCRYPT));                                    │
└──────────────────────────────────────────────────────────────┘

COMO TROCAR A SENHA DE ADMIN:

OPÇÃO 1 - Direto no código (Recomendado):

┌──────────────────────────────────────────────────────────────┐
│ // Defina sua senha aqui                                    │
│ $minhaSenha = 'MinhaS3nhaF0rt3!';                           │
│                                                              │
│ define('ADMIN_USERNAME', 'admin');                           │
│ define('ADMIN_PASSWORD', password_hash($minhaSenha,         │
│        PASSWORD_BCRYPT));                                    │
└──────────────────────────────────────────────────────────────┘

OPÇÃO 2 - Gerar hash externamente:

1. Acesse: https://bcrypt-generator.com/
2. Digite sua senha
3. Copie o hash gerado
4. Cole no código:

┌──────────────────────────────────────────────────────────────┐
│ define('ADMIN_PASSWORD', '$2y$10$abcd...seu_hash_aqui');    │
└──────────────────────────────────────────────────────────────┘

🔒 DICAS DE SEGURANÇA:

✅ Use senhas FORTES:
   • Pelo menos 12 caracteres
   • Letras maiúsculas e minúsculas
   • Números
   • Caracteres especiais (!@#$%^&*)
   
❌ NÃO use senhas fracas:
   • 123456
   • password
   • admin
   • naipersclub
   • datas de nascimento

✅ NUNCA compartilhe:
   • config/database.php
   • Suas senhas
   • Acesso ao cPanel

✅ Faça backup:
   • Salve config/database.php em local seguro
   • Anote suas credenciais em gerenciador de senhas
   • Nunca deixe no código versões antigas

❓ PERGUNTAS FREQUENTES:

Q: Onde pego as credenciais MySQL?
A: No cPanel → MySQL Databases (onde você criou o banco)

Q: Como recupero a senha de admin se esquecer?
A: Edite config/database.php e defina uma nova

Q: Posso mudar o username "admin"?
A: Sim! Altere ADMIN_USERNAME no arquivo

Q: Preciso criar vários usuários admin?
A: Não, este sistema tem apenas 1 admin

Q: As senhas ficam salvas onde?
A: MySQL password está no config/database.php
   Admin password está hasheada no mesmo arquivo
