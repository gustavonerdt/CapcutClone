📦 CONTEÚDO DESTA PASTA:
   Todos os arquivos prontos para upload no Hostinger shared hosting

🚀 COMO INSTALAR:

1. UPLOAD DOS ARQUIVOS:
   - Faça upload de TODOS os arquivos desta pasta para public_html/
   - Mantenha a estrutura de pastas intacta

2. CONFIGURAR BANCO DE DADOS:
   - Crie um banco MySQL no painel do Hostinger
   - Importe o arquivo: schema_mysql.sql
   - Anote: nome do banco, usuário e senha

3. CONFIGURAR CONEXÃO:
   - Edite o arquivo: config/database.php
   - Coloque suas credenciais MySQL:
     * DB_HOST (geralmente localhost)
     * DB_NAME
     * DB_USER
     * DB_PASS

4. TESTAR:
   - Acesse: seudominio.com
   - O quiz deve funcionar normalmente
   - Teste o formulário de cadastro

5. ACESSAR ADMIN:
   - URL: seudominio.com/admin/
   - Usuário: admin
   - Senha: naipersadmin2024
