📋 PRÉ-REQUISITOS:

✅ Conta Hostinger ativa
✅ Acesso ao cPanel

✅ PHP 7.4 ou superior
✅ MySQL 5.7 ou superior

══════════════════════════════════════════════════════════════════

🚀 PASSO 1 - CRIAR BANCO DE DADOS

1. Entre no cPanel do Hostinger

2. Vá em "MySQL Databases"

3. Crie um novo banco:
   Nome: naipers_club (ou qualquer outro)
   
4. Crie um usuário MySQL:
   Usuário: (escolha um nome)
   Senha: (senha forte!)
   
5. Associe o usuário ao banco:
   - Selecione o usuário criado
   - Selecione o banco criado
   - Marque "ALL PRIVILEGES"
   - Salvar

6. ANOTE ESTAS INFORMAÇÕES:
   ✏️ Nome do banco: _____________________
   ✏️ Usuário MySQL: _____________________
   ✏️ Senha MySQL: _______________________

══════════════════════════════════════════════════════════════════

📤 PASSO 2 - UPLOAD DOS ARQUIVOS

OPÇÃO A - Via File Manager (Recomendado):

1. cPanel → File Manager

2. Navegue até "public_html/"

3. Upload de TODOS os arquivos desta pasta:
   - Selecione tudo
   - Arraste para o File Manager
   - Ou use botão "Upload"

4. Aguarde upload completar

OPÇÃO B - Via FTP:

1. Use FileZilla ou outro cliente FTP

2. Conecte no seu Hostinger

3. Envie todos os arquivos para public_html/

══════════════════════════════════════════════════════════════════

💾 PASSO 3 - IMPORTAR BANCO DE DADOS

1. cPanel → phpMyAdmin

2. Selecione o banco criado (naipers_club)

3. Clique em "Import" (Importar)

4. Escolha o arquivo: schema_mysql.sql

5. Clique em "Go" (Executar)

6. Aguarde mensagem de sucesso ✅

7. Verifique se 3 tabelas foram criadas:
   - sessions
   - tracking_events  
   - quiz_responses

══════════════════════════════════════════════════════════════════

⚙️ PASSO 4 - CONFIGURAR CREDENCIAIS

1. Abra o arquivo: config/database.php

2. Edite com suas informações do PASSO 1:

   <?php
   // Configuração do Banco MySQL
   define('DB_HOST', 'localhost');
   define('DB_NAME', 'seu_usuario_naipers_club');  ← TROCAR
   define('DB_USER', 'seu_usuario_mysql');         ← TROCAR
   define('DB_PASS', 'sua_senha_mysql');           ← TROCAR
   
   // Configuração de Admin
   define('ADMIN_USERNAME', 'admin');  
   
   // 🔴 IMPORTANTE: TROCAR SENHA!
   $novaSenha = 'SUA_SENHA_SEGURA_AQUI';
   define('ADMIN_PASSWORD', password_hash($novaSenha, PASSWORD_BCRYPT));
   ?>

3. Salve o arquivo

4. IMPORTANTE: Não compartilhe este arquivo!

🔒 SEGURANÇA - ANTES DE DIVULGAR:

1. ✅ Trocar senha de admin
   Arquivo: config/database.php

2. ✅ Ativar HTTPS
   Arquivo: .htaccess
   Descomentar linhas:
   # RewriteCond %{HTTPS} off
   # RewriteRule ^(.*)$ https://%{HTTP_HOST}%{REQUEST_URI} [L,R=301]

3. ✅ Fazer backup do banco
   phpMyAdmin → Export → Salvar .sql

🐛 TROUBLESHOOTING (Se algo não funcionar):

PROBLEMA: "Database connection failed"
SOLUÇÃO: 
  • Verifique credenciais em config/database.php
  • Confirme que o banco existe no phpMyAdmin
  • Teste conexão MySQL no cPanel

PROBLEMA: Página em branco
SOLUÇÃO:
  • Ative exibição de erros
  • Verifique logs de erro no cPanel
  • Confirme que PHP 7.4+ está ativo

PROBLEMA: "404 Not Found" nas APIs
SOLUÇÃO:
  • Verifique se .htaccess está na raiz
  • Confirme mod_rewrite ativo
  • Teste acesso direto: /api/sessions.php

PROBLEMA: Admin não faz login
SOLUÇÃO:
  • Verifique senha em config/database.php
  • Use password_hash('sua_senha', PASSWORD_BCRYPT)
  • Limpe cache do navegador

PROBLEMA: Quiz não avança
SOLUÇÃO:
  • Abra Console do navegador (F12)
  • Veja se há erros JavaScript
  • Verifique se assets/ carregaram
